# 説明
# m: メルケプストラム次数（実際はパワー項を追加してm+1次元ベクトルになる）
# a: all-pass constant
# e: メルケプストラム抽出時にピリオドグラムに足しこむ微小値
# K: GMMのコンポーネント数
# channel: 変換したいWavファイルのチャンネル数
# frame_rate: 変換したいWavファイルのサンプリング周波数[Hz]
# exclude_both_mcep_ends: 2者のメルケプストラムから対応関係を学習するときに、メルケプストラム波形の先端と終端を除外するか（0以外を指定すると、先端および終端に一定値が続く場合に学習データから除去される）
